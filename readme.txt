== Description ==

Optimize your post(s) and page(s) for the Search Engines using this FREE and effective seo plugin!
 
= SEO Plugin Features =

* Auto-check current post
* Auto-check timing – (seconds)
* Title/separator (default – )
* Seo Title - position (Admin post listing)
* Show's SEO score
* Advanced SEO rules and settings.
* Keyword in title tag 
* Keyword in alt text
* Should describe graphic
* Keyword Density 
* Individual keyword density 6% – (each keyword/total words)
* Keywords in H1, H2, H3
* Keywords in font style - In strong, bold, italic, strong.

== Installation ==

1. Extract the `seo-plugin`-folder and transfer it to the `/wp-content/plugins/` directory
1. Activate it through the 'Plugins' menu in your WP Admin.
1. Go to Seo Plugin page from your wp panel to adjust settings and rules.
